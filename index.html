Quick Tips:

  • Use the Tab key to auto-complete commands and filenames.
  • Press Ctrl+C to cancel a running command.
  • Use the ↑ / ↓ arrow keys to navigate through command history.
  • Command names are case-sensitive on macOS and Linux, but not on Windows.
  • File and folder names are case-sensitive in Linux; in macOS and Windows, names are not case-sensitive.
  • Use the --help flag with most commands to view available options and usage.
  • Use forward slashes (/) in macOS/Linux paths, and backslashes (\) in Windows paths (e.g., cd Documents/ vs cd Documents\).
